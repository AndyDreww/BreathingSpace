A Warm Welcome

Hello yogis! We are a yoga studio based in the center of Nottingham. We practice various styles of yoga.

We are a beginner friendly studio, so come have a chat with us to find out more - we're sure you'll feel welcomed!

Classes

Beginners Yoga

  • This class will ease you into the practice. You will feel welcomed and be immersed into the benefits of yoga.
  • This class will be supportable for anyone and everyone, regardless of age or level of fitness.
  • This class is a great introduction to yoga poses. You will soon start feeling the benefits of yoga.
£8 a class / £25 a month Join Class

Iyengar Yoga

  • This class is a guided and precise form of yoga. The focus is on alignment.
  • You may need a little experience in yoga for this class, however, the instructor is very involved in this class and will help with any incorrect form.
  • The focus of this class is to realign your body. Props may be used during the class to help yogis will the correct form.
£8 a class / £25 a month Join Class

Vinyasa Yoga

  • This class focuses on coordinate movement, moving from one pose to another flawlessly. No two classes will be the same as we vary the poses each time!
  • This class is geared towards more advanced yoga practitioners as you need familiarity with various yoga poses.
  • Given the variety of poses, this class helps with body posture and having a more balanced body.
£8 a class / £25 a month Join Class
